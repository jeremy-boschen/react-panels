# Security Policy

## Supported Versions

We release patches for security vulnerabilities. Which versions are eligible for receiving such patches depends on the CVSS v3.0 Rating:

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please report them via GitHub's security advisory feature:

1. Go to https://github.com/jeremy-boschen/react-adjustable-panels/security/advisories/new
2. Click "Report a vulnerability"
3. Fill out the form with details about the vulnerability

You can also email security concerns to: (add your email if you want)

### What to Include

Please include the following information in your report:

- Type of vulnerability
- Full paths of source file(s) related to the vulnerability
- The location of the affected source code (tag/branch/commit or direct URL)
- Any special configuration required to reproduce the issue
- Step-by-step instructions to reproduce the issue
- Proof-of-concept or exploit code (if possible)
- Impact of the vulnerability, including how an attacker might exploit it

### Response Timeline

- We will acknowledge your report within 7 days
- We will provide a detailed response within 21 days indicating the next steps
- We will work on a fix and coordinate disclosure with you
- Once a fix is ready, we will release a security update
- We will publicly disclose the vulnerability after the fix is released

## Best Practices

When using @jeremy-boschen/react-adjustable-panels:

- Always use the latest version
- Review the changelog for security updates
- Enable Dependabot alerts in your repository
- Follow React security best practices

## Disclosure Policy

- We follow coordinated vulnerability disclosure
- We will work with reporters to understand and address issues
- We will publicly disclose vulnerabilities after patches are released
- Security advisories will be published on GitHub

Thank you for helping keep @jeremy-boschen/react-adjustable-panels and its users safe!
